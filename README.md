# CHIP-8 Emulator

Yet another CHIP-8 emulator.

## Build

To build the project you will need SDL2 installed as a dependency for this project.

```bash
mkdir build
make
```

## Controls
```
Emulator Keybinds
-----------------------
QUIT           | Escape
PAUSE / RESUME | Space
Reset          | Return
```

```
CHIP-8 Machine Keybinds
-------------------------------
CHIP-8 Keypad | QWERTY Keyboard
123C          | 1234
456D          | QWER
789E          | ASDF
A0BF          | ZXCV
```
